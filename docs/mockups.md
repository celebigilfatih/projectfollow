# Yenilenmiş UI Mockupları

## Proje Özeti (Tek Satır)
- Sol: Proje açıklaması (truncate)
- Sağ: Durum rozeti, Toplam, Tamamlanan, %Tamamlanma, Yaklaşan tarih
- Kart arka plan: açık/kapalı tema için yumuşak gradient

## Görevler Tablosu
- Sütunlar: [Seçim], Görev, Durum, Öncelik, Atanan, Son tarih, Aksiyon
- Satır Hover: arka plan hafif koyulaşır, gölge belirginleşir
- Aksiyon Menüsü: Durum/Öncelik/Atama/Son tarih hızlı düzenleme

## Filtre Çubuğu (Tek Satır)
- Arama Input (esneyen genişlik)
- Hızlı filtreler: “Sadece bana atanan”, “Son tarihi geçen”
- Select: Durum, Öncelik (sabit genişlik)
- Butonlar: Uygula, Temizle

## Yeni Görev Ekle
- Hızlı mod: Başlık + Ekle
- Detaylar: Kişi, Takım, Son tarih, Çoklu kişi atama

## Tema Toggle
- Sağ üst; Sun/Moon ikon; localStorage ve `prefers-color-scheme` uyumu
